# Knowledge Graph: Smooth Manifolds

## L1: Core Definitions
- Chart: local coordinate system (index, dim, boundaryDim)
- Atlas: collection of compatible charts (charts, dimension, name)
- SmoothManifold: typeclass with dim, atlas, topological properties
- SmoothStructure: equivalence class of atlases
- TransitionMap: map between chart domains
- Orientation: positive/negative orientation
- PartitionOfUnity: subordinate to open cover

## L2: Core Concepts
- Cocycle condition: triple overlap consistency
- Maximal atlas: contains all compatible charts
- Compatibility of charts
- Manifold with boundary: modeled on half-space
- Orientability: positive Jacobian determinants
- Paracompactness: existence of partitions of unity

## L3: Math Structures
- Category Mfd: objects = smooth manifolds, morphisms = smooth maps
- Product manifolds: M x N
- Quotient manifolds: M/G (free proper actions)
- Submanifolds: embedded, immersed, initial
- Homogeneous spaces: G/H
- Diffeomorphism group Diff(M)
- Smooth structures in low dimensions

## L4: Fundamental Theorems
- Inverse Function Theorem on manifolds
- Implicit Function Theorem
- Rank Theorem (constant rank)
- Whitney Embedding (M^n -> R^{2n})
- Whitney Immersion (M^n -> R^{2n-1})
- Sard's Theorem (regular values dense)
- Preimage Theorem (regular level sets)
- Morse Inequalities
- Poincare-Hopf Index Theorem
- Stokes' Theorem

## L5: Proof Techniques
- Chart-based arguments
- Partition of unity arguments
- Transversality methods
- Compactness arguments
- Flow box (straightening)
- Morse theoretical methods
- Surgery arguments

## L6: Canonical Examples
- R^n (Euclidean space)
- S^n (sphere, orientable)
- RP^n (real projective, orientable iff n odd)
- CP^n (complex projective, always orientable)
- T^n (n-torus)
- Lie groups: GL(n), SL(n), O(n), SO(n), U(n), SU(n)
- Grassmannians: Gr(k,n)
- Lens spaces
- Exotic spheres (S^7: 28 structures)
